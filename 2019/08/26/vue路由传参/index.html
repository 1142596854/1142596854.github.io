vue路由传参

发表于 2019-08-26 | 分类于 vue

参考

route和router的区别

query通过path切换路由，params通过name切换路由

query通过this.$route.query接受，params通过this.$route.params接受

// query通过this.$route.query接收参数 created () { const id = this.$route.query.id; } // params通过this.$route.params来接收参数 created () { const id = this.$route.params.id; }

展示方式

query传参的url展现方式：/detail?id=1&user=123&identity=1&更多参数

params＋动态路由的url方式：/detail/123

params动态传参必须在路由中定义，且跳转时必须加上参数，否则为空白画面

{
path: ‘/detail/:id’,
name: ‘Detail’,
component: Detail
},

params刷新后会消失

route和router的区别

router是vue的一个实例

通过Vue.use(VueRouter)和VueRouter构造函数得到一个router的实例对象，这个对象是一个全局对象。它包含了所有的路由以及许多关键的对象和属性。

  • push

push方法其实和是等同的。
注意：push方法的跳转会向 history 栈添加一个新的记录，当我们点击浏览器的返回按钮时可以看到之前的页面。

  • go
  • replace

route是一个路由对象

每一个路由都会有route对象，它是一个局部对象。$route对象表示当前的路由信息，包含了当前 URL 解析得到的信息。包含当前的路径，参数，query对象等

  • path
  • params
  • query
  • hash
  • fullPath
  • matched
  • name
  • meta
