各种定位

发表于 2019-10-13

display

参考

display：none

不会占据空间，不会显示，主要用于改善重排和重绘

display：inline

将元素设置为行内元素。设置了以后再设置元素的高度和宽度都无效，text-align也无效，设置了line-height会让line元素居中

display：block

设置为块级元素。默认宽度继承父亲宽度，独占一行。高度默认以子元素撑开的高度为准，也可以指定高度

display：list-item

将元素作为列表显示

display：inline-block

设置为行内块元素。可以自己设置宽高，可多个占一行

可利用text-align:justify实现固定款高的列表两端对齐布局（行内元素的总宽度至少占一行，宽度不足一行时没有用）

display:table

见table和flex

display：flex

见table和flex

position

static

无特殊定位，遵循文档流

relative

相对定位是相对对象出于标准文档流中的位置依据top和left定位。top和left并不改变对象在文档流中的占位空间。

marging和padding会改变对象在标准文档流中的占位空间

absolute

脱离了标准文档流，相对于static定位以外的第一个父元素，使用left和top进行绝对定位。（使用absolute的时候必须指定left，top，right和bottom中的至少一个）

实质上是相对于static定位以外的第一个父元素的border进行定位

fixed

脱离了文档流，以窗口为原点进行偏移定位，不会随着滚动条的滚动而偏移

z-index

两个同级元素拥有同样的值，那么将依据他们在文档六中的顺序层叠

父子关系不能用z-index设定上下关系

float

参考链接

浮动元素会从普通文档流中脱离，会影响周围元素的对齐进行焕然

让block元素无视float元素，让inline元素流水一样环绕着float团苏来实现浮动布局

浮动会向左向右浮动，直到碰到另一个浮动元素

元素特性

块级框

元素被设置浮动后，浮动元素会生成一个块级框，可以设置他的宽和高

默认情况下，div元素的宽是100%，一旦让该div浮动起来，立刻会像inline元素一样产生包裹性，宽度会跟随内容自适应。

行内元素的例子：

当span设置float属性后，可以指定width，height值。 同时可以发现，当多个元素指定为float且同时为left/right时，元素是紧挨着排列，行内宽度不够时再换行排列。

行内元素设置float以后可以近似认为是具有inline-block的属性，可以设置宽高等

因为img设置了浮动以后脱离了文档流，div2则认为里面没有元素，发生了高度坍塌
